# Bricks Calc — iOS UI Kit

Mortgage calculator app. Source: `fredjdc/Bricks-Calc` (SwiftUI, includes `AppTheme.swift`). Accent = blue `#007AFF`.

Core screens:
- **Calculator** — primary input form (home price, down, rate, term) with live monthly payment
- **Breakdown** — principal / interest / PMI / taxes split
- **Scenarios** — multiple saved scenarios for side-by-side comparison
