# Bricks Website — UI Kit

Marketing site recreation for the **Bricks umbrella** page. Source: `fredjdc/Bricks-Page` (`styles.css`, `index.html`).

Visual DNA: full-bleed editorial hero photograph (desaturated, grained), Soft-Emboss cards on the cool-neutral canvas, floating pill nav, heavy display type.

## Components
- `FloatingHeader.jsx` — glass pill nav with logo + globe
- `Hero.jsx` — full-bleed image hero with short-stack copy and primary CTA
- `ProductGrid.jsx` — 3-up grid of Scan / Calc / Leads product cards
- `FeatureRow.jsx` — trio-list feature layout per product
- `DownloadCTA.jsx` — App Store button card
- `Footer.jsx` — minimal footer with social icons

## Screens
- `index.html` — full umbrella landing: hero → "Three tools. One suite." grid → per-product feature section → footer.
