# Bricks Leads — iOS UI Kit

CRM for real estate agents. Source: `fredjdc/Bricks-Leads` (SwiftUI with `MeshGradients.swift`). Accent = orange `#FF9500`.

Core screens:
- **Leads** — prioritized list of active leads with status badges
- **LeadDetail** — single-lead view with timeline, next follow-up, linked property
- **Schedule** — day view with appointments and touch-list
