# Bricks Website Repo Map

Fast map of the website repository.

## Main Areas

| Path | Purpose |
|---|---|
| `index.html` and `*.html` pages | site pages |
| `styles.css` | global stylesheet |
| `scripts.js` | global JavaScript |
| `hero-background.js` | hero animation |
| `images/` and `images-*` | shared and product-specific assets |
| `.well-known/` | universal link support files |
| `sitemap.xml` | SEO sitemap |
